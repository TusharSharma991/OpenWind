## Phase 3 — Metabase embed

**Goal:** Metabase docker service, signed embed token API, iframe panel in admin dashboard. Can ship after pilot onboarding begins — not a pilot blocker.

**Gate:** `POST /reporting/embed-token` returns valid signed URL; iframe renders in dashboard with tenant filter applied; graceful fallback when env vars absent.

| task | description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         | requirement | status |
| ---- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------- | ------ |
| T22  | **Metabase docker service** — add `metabase` service to `docker-compose.yml`: image `metabase/metabase:latest`, port `3030:3000`, depends on `db`, env vars `MB_DB_TYPE=postgres MB_DB_CONNECTION_URI=...`. Add `METABASE_SECRET_KEY` (string, optional) and `METABASE_SITE_URL` (url, optional) and `METABASE_DASHBOARD_ID` (coerce number, optional) to `packages/config/src/env.ts` — all optional, feature disabled if absent. Add vars to `.env.example`.                                                                      | R7          | todo   |
| T23  | **Embed token API** — `apps/api/src/routes/reporting/embed-token.ts`. `POST /reporting/embed-token`. Requires `admin` role. Guard: if `METABASE_SECRET_KEY` not configured → 503 `REPORTING_NOT_CONFIGURED`. Sign JWT using `jsonwebtoken` (or `jose`) with HS256, payload `{ resource: { dashboard: METABASE_DASHBOARD_ID }, params: { tenant_id: tenantId }, exp: now+600 }`. Return `{ iframeUrl: "${METABASE_SITE_URL}/embed/dashboard/<token>#bordered=true&titled=true", expiresAt: ISO }`. Rate-limit 30 req/min per tenant. | R7          | todo   |
| T24  | **Admin-ui dashboard iframe panel** — in `apps/admin-ui/src/pages/dashboard.tsx`, add "Analytics" panel section. On mount: call `POST /reporting/embed-token`. Render `<iframe src={iframeUrl}>` in a fixed-height card (500px). Set up `setInterval` at 9 min to refresh token and update `src`. On 503 response: render "Reporting not configured" placeholder card instead of iframe. Loading spinner while token is fetching.                                                                                                   | R7          | todo   |
